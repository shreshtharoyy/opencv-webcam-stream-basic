# OpenCV Webcam Stream (Basic)

## Description
This project opens the system's default webcam using OpenCV and displays the live video feed in a window.

The webcam stream runs in a loop and closes when the user presses the 'q' key.

---

## Requirements
- Python installed
- OpenCV library

Install OpenCV using:
pip install opencv-python

---

## How to Run

1. Open terminal in the project folder.

2. Run the following command:
python webcam_stream.py

3. A webcam window will open showing live video feed.

4. Press 'q' on the keyboard to close the webcam window.
